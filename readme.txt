== Description ==

The YourCareEverywhere content widget allows you to display content from YourCareeverywhere.com to visitors on your website for free. The widget automatically refreshes as YourCareEverywhere.com is refreshed with new articles, content, and health and wellness information. 

### Shortcode Parameters
YourCareEverywhere Feed accepts 4 shortcode parameters for overriding the default settings:
* title * - Adds a custom title above the feed 
* format * - displays either a vertical or horizontal feed
* height * - height of the feed
* width * - width of the feed window (only used on vertical feeds)

== Installation ==

Installing "YourCareEverywhere Feed" can be done either by searching for "YourCareEverywhere Feed" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
